## Development

Make sure to install Deno:
<https://deno.land/manual/getting_started/installation>.

Run the project locally:

```sh
deno task start
```

Format the project:

```sh
deno fmt
```

Check for common errors:

```sh
deno lint
deno check main.ts
```
